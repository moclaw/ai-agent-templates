# Multi-Language Support

This template supports multiple programming languages. Choose the languages you want to work with during setup.

## Supported Languages

- **JavaScript/TypeScript** - Node.js applications
- **Python** - Python 3.8+ applications
- **Go** - Go 1.19+ applications
- **Java** - Java 17+ applications
- **C#/.NET** - .NET 8+ applications
- **Ruby** - Ruby 3.0+ applications

## Setup

Run the interactive setup script:

```bash
./scripts/setup.sh
```

Select "Multiple Languages" when prompted for language selection.
